# 2026-05-21 工作记忆

## 配色减重优化

用户反馈配色过重，进行了全局配色调整，commit `97eecc0`。

**主要变更**：
- Header 渐变：`#07553b` → `#1b6b48 ~ #2d8b61 ~ #52b888`（提亮降饱和）
- 要点速览标题栏：深绿实色块 → 浅绿背景 `#f0f7f3` + 绿色文字（最显著变化）
- 表格表头：深绿底白字 → 浅灰绿底深色字
- 条目左侧竖条：4px `#2ecc87` → 3px `#86c9a8`（更柔和）
- 条目背景：`#f8fdfb` → `#fafcfb`（接近纯白）
- 链接按钮：改为 opacity 0.85，hover 全实
- 页面背景：蓝灰 `#f0f4f8` → 轻绿灰 `#f5f7f5`

**用户偏好记录**：配色整体偏轻、降饱和，避免大面积深色实色块。

## 搜索框 + 标签筛选功能上线

commit `38fc510`，新增两个交互功能：

**搜索框**（Tabs 下方搜索栏）
- 实时全文搜索当前期次所有条目，匹配词高亮（黄色 mark）
- 右侧显示"共 N 条"计数；无结果时显示提示语
- 搜索/筛选状态在切换 Tab 时自动重置

**标签筛选**（7个 filter tag）
- 全部 / 新功能 / API更新 / 规则 / 治理公告 / 活动激励 / 专项治理
- 筛选后概览卡片自动隐藏，聚焦条目内容

**技术要点**：build_html.js 模板字符串中 RegExp 反斜杠转义问题，最终用 `.split('').map().join()` 绕开字符类括号冲突。

## 截止日期提醒 + 分享单条功能上线

commit `77fbfb2`

**截止日期提醒栏**（Header 正下方黄色 banner）
- 自动扫描所有期次 `.item.alert` 和 `.item.notice` 中的日期（支持"X月X日""YYYY年X月X日"格式）

- 点击复制「标题\n链接」两行格式（去掉 tag/badge 标签文字，只保留纯标题）
- 无链接条目（本期无更新等）不显示按钮
